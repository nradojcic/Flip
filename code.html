The Code for FLIP 


                // get a string from the page
                // Controller funcyion
                function getValue() {
                  
                  // hide the alert box at the beginning of the program
                  document.getElementById("alert").classList.add("invisible");

                  let userString = document.getElementById("userString").value;

                  let revString = reverseString(userString);

                  displayString(revString);
                }

                // reverse the string
                // Logic function
                function reverseString(userString) {
                  let revString = [];

                  // reverse the string using a for loop, backwards
                  for (let i = userString.length - 1; i >= 0; i--) {
                    
                    revString += userString[i];
                  
                  }

                  return revString;
                }

                // display the reversed string on the page
                // View function
                function displayString(revString) {
                  
                  // write to the page
                  document.getElementById("msg").innerHTML = 
                    `Your string reversed is: <strong>${revString}</strong>`;

                  // show the alert box
                  document.getElementById("alert").classList.remove("invisible");
                }

The Code is structured in three functions.

1) getValue function

Function getValue is the Controller function. It is triggered by an onClick event connected to a button on the App page.

It first makes sure the alert display box is hidden, by adding the invisible CSS class to it.

It then collects text from the input field on the page, and routes the request to other functions.

Then it calls reverseString function to reverse the collected string, and then the displayString function to show the alert box and display the reversed string.

2) reverseString function

Function reverseString is the Logic function. It reverses the user's string.

It accepts userString as the argument, and then runs a FOR loop BACKWARDS from the end towards the beginning of the string, adding values one by one to revString string. The function then returns the string.

3) displayString function

Function displayString is the View function. It displays the reversed string on the web page.

It accepts a string as the argument. And then uses template string to format the output sentence inserting in it revString value. It then shows the message on the screen by removing invisible CSS class from the alert box that was made hidden at the beginning of the program.
